# PDF简历转交互式HTML — 使用手册

将 PDF 简历自动转换为带选项卡切换的交互式 HTML 网页，支持在线分享、打印和 PDF 下载。

---

## 目录结构

```
D:\AI\PDFResume2HTML\
├── generate_resume.py   # 核心脚本：PDF → HTML
├── 一键更新.bat          # 一键生成 + 自动推送 GitHub
├── resume.html           # 生成的HTML简历（本地预览）
├── index.html            # 短链接跳转入口
└── 使用手册.md           # 本文件
```

---

## 快速开始

### 1. 安装依赖（首次使用）

打开终端，执行：

```bash
pip install pdfplumber
```

### 2. 编辑 PDF 路径

用记事本打开 `generate_resume.py`，修改第 27 行：

```python
PDF_PATH = r"你的PDF文件路径.pdf"
```

### 3. 生成 HTML

双击 `一键更新.bat`，等待执行完成。生成的 `resume.html` 可双击在浏览器中直接预览。

## HTML 功能说明

| 功能 | 操作方式 |
|------|----------|
| 选项卡切换 | 点击顶部导航「职业概述 / 教育背景 / 技术技能 / 工作经历 / 项目经验 / 证书资质」 |

打印和 PDF 下载时会自动展开所有模块，按选项卡顺序输出，隐藏导航和工具栏，排版整洁。

## 常见问题

**Q: 生成的内容不全或错位？**

将 `generate_resume.py` 第 29 行改为 `DEBUG = True`，重新运行后会生成 `debug_raw.txt`，可查看 PDF 提取的原始文本用于诊断。

**Q: PDF 下载不生效？**

首次需联网加载 html2pdf.js 组件（约 200KB），加载后会自动缓存。离线环境会自动降级为「打印 → 另存为 PDF」。

**Q: 如何自定义样式？**

编辑 `generate_resume.py` 中 `build_html()` 函数内的 `<style>` 块，修改 CSS 变量即可更换主题色。
